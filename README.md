# 🦺 CoW Log — Clerk of Works Site Journal

Professional site documentation app for Clerks of Works, Resident Engineers, and Site Inspectors.

## Features

### 📔 Detailed Site Diary
- Activity type classification (Work in Progress, Material Delivery, Inspection, Meeting, Delay/Issue)
- Labour count tracking
- Equipment on site
- Materials delivered log
- Issues & concerns flagging
- Contractor personnel tracking

### ✅ Inspection Checklists
- Concrete Pour (Pre-Pour) — 9 items
- Steel Reinforcement — 8 items
- Blockwork/Masonry — 8 items
- Excavation & Earthworks — 7 items
- Waterproofing — 8 items
- Plastering & Rendering — 8 items
- Each item with pass/fail + comment
- Auto-generates pass/fail result
- Corrective action field

### 🚨 NCR System
- Minor / Major / Critical severity
- Responsible party tracking
- Root cause analysis
- Closeout verification
- Open NCR dashboard

### 📋 Site Instructions
- Issued to contractor tracking
- Drawing/specification reference
- Deadline field
- Acknowledge status

### 📄 Auto-Generated Professional Report
9-section comprehensive daily report:
1. Work Progress & Observations
2. Inspections & Tests
3. Non-Conformance Reports
4. Site Instructions
5. Plant & Equipment on Site
6. Labour on Site
7. Materials Delivered
8. Health & Safety Observations
9. General Remarks

Export options: Copy, Save, Share

### 🔧 Technical
- Offline-first — all data in localStorage
- No server, no database, no internet needed
- PWA — install to home screen
- Dark theme optimized for site use
- Single HTML file — no dependencies

## Usage
1. Open in any browser
2. Add to Home Screen for app-like experience
3. Start logging diary entries, inspections, NCRs
4. View auto-generated report under "Report" tab
5. Copy/share report daily

Built for construction professionals in Kenya and beyond.
